Este repositório contém as resoluções dos exercícios do TPC1.
Cada exercício foi implementado num ficheiro individual para facilitar a leitura e o teste.
- ex1.py - reverse(n) - Inverte a string utilizando slicing [::-1]
- ex2.py - countA(n) - Conta ocorrências de 'a' e 'A' usando loops; countA2(n) - Conta ocorrências de 'a' e 'A' usando o método .count()
- ex3.py - countVowels(n) - Identifica o número total de vogais em letra procurando as vogais da palavra, que foi posta em texto minúsculo usando o .lower(), na lista de vogais dada [a,e,i,o,u]. 
- ex4.py - lower(n) - Converte o texto para minúsculas com .lower()
- ex5.py - upper(n) - Converte o texto para maiúsculas com .upper()
- ex6.py - capicua(n) - Verifica se a string é igual à sua versão invertida utilizando o slicing [::-1]
- ex7.py - balanceado(s1, s2) - Verifica se todos os caracteres de s1 estão presentes em s2
- ex8.py - ocorrencia(s1, s2) - Conta quantas vezes a substring s1 aparece em s2
- ex9.py - anagrama(s1, s2) - Verifica se a lista de letras da palavra s1 organizada por ordem alfabética é a mesma da palavra 2, returnando True se for igual (um anagrama) ou False se não for.
